#### 链表

##### 添加操作 O(n)

* addLast(e) O(n)
* addFirst(e) O(1)
* add(index, e) O(n/2) = O(n)

##### 删除操作 O(n)

* removeLast(e) O(n)
* removeFirst(e) O(1)
* remove(index, e) O(n/2) = O(n)

##### 修改操作 O(n)

* set(index, e) O(n/2) = O(n)

##### 查找操作 O(n)

* get(index) O(n)
* contains(2) O(n)

> 如果只对链表头进行操作，复杂度为 O(1)
